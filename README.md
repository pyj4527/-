# Seoulody Backend

서울 장소 기반 음악 추천 서비스 백엔드입니다.

## 기술 스택

- Python
- Django
- Django REST Framework

## 실행 방법

### 1. 가상환경 생성

python -m venv .venv

### 2. 가상환경 실행

Windows:

.venv\Scripts\activate

Mac:

source .venv/bin/activate

### 3. 패키지 설치

pip install -r requirements.txt

### 4. 서버 실행

python manage.py runserver

## 주요 기능 예정

- 장소 API
- 노래 CRUD API
- 장소별 노래 추천 API
